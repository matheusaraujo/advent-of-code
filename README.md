## running

#### all commands
```bash
$ aoc help
```
#### starting a new day
```bash
$ aoc create perl
```

#### running
```bash
$ aoc run
```

#### watch mode
```bash
$ aoc run --watch
```

#### preparing commit
```bash
$ aoc commit
```
