### running

``` {.bash}
# help
$ make help

# starting
# supported langs: perl, python
$ make create year=2000 day=00 lang=perl

# running
$ make run year=2000 day=00

# linting
$ make lint year=2000 day=00

# analysis
$ make analysis year=2000 day=00
```
