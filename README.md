### running

```bash
# help
$ make help

# starting
# supported langs: perl, python
$ make create year=1988 day=00 lang=perl

# running
$ make run year=1988 day=00

# linting
$ make lint year=1988 day=00

# analysis
$ make analysis year=1988 day=00
```
